# 🎨 Design Tokens - Actuaría System

**Sistema de tokens sincronizados desde Figma**  
*Última actualización: Abril 2024*

---

## 📖 Introducción

Este Design System proporciona un conjunto exhaustivo de **Design Tokens** que garantizan coherencia visual y funcional en toda la plataforma de Actuaría. Los tokens están disponibles en **3 formatos**:

- **CSS Variables** → Para estilos directos en HTML/CSS
- **TypeScript** → Para importación en React/Next.js con type-safety
- **Figma Variables** → Single source of truth en diseño

## 🎨 Paleta de Colores

### Colores Primarios

Los colores primarios (azul) se utilizan para acciones principales, enlaces y elementos de marca.

| Nivel | Valor | Uso |
|-------|-------|-----|
| **50** | `#E8F2FF` | Fondos muy claros |
| **100** | `#D0E5FF` | Hover states ligeros |
| **200** | `#A2CBFF` | Estados hover |
| **300** | `#73B1FF` | Estados activos ligeros |
| **400** | `#4597FF` | Elementos secundarios |
| **500** | `#1F7FFF` | **PRIMARY** - Acciones principales |
| **600** | `#1563D6` | Hover activo |
| **700** | `#0D47AD` | Estados activos/pressed |
| **800** | `#082B84` | Énfasis fuerte |
| **900** | `#04165B` | Texto oscuro sobre primario |

**Uso en CSS:**
```css
.button-primary {
  background-color: var(--color-primary-500);
  color: var(--color-primary-900);
}

.button-primary:hover {
  background-color: var(--color-primary-600);
}
```

**Uso en React/TypeScript:**
```typescript
import { colors } from '@/tokens';

const buttonStyles = {
  backgroundColor: colors.primary[500],
  '&:hover': {
    backgroundColor: colors.primary[600],
  },
};
```

## 📝 Tipografía

### Font Families

```typescript
const fontFamily = {
  sans: "Inter, -apple-system, BlinkMacSystemFont, ...",  // UI Default
  serif: "Merriweather, Georgia, serif",                  // Editorial
  mono: "Fira Code, Courier New, monospace",              // Code
};
```

**Stack recomendado:**
- **UI:** Inter (sans) - Interfaz limpia y profesional
- **Contenido:** Merriweather (serif) - Documentos, reportes
- **Código:** Fira Code (mono) - Ejemplos, valores técnicos

### Composiciones Tipográficas Predefinidas

Para mayor consistencia, utiliza estas composiciones en lugar de valores individuales:

```typescript
import { typographyComposites } from '@/tokens';

// Display
typographyComposites.displayLarge    // 44px, Bold, leading 52px
typographyComposites.displaySmall    // 36px, Bold, leading 44px

// Headings
typographyComposites.headlineLarge   // 30px, Bold, leading 36px
typographyComposites.headlineSmall   // 24px, Bold, leading 32px

// Títulos
typographyComposites.titleLarge      // 20px, Semibold, leading 28px
typographyComposites.titleSmall      // 18px, Semibold, leading 26px

// Body
typographyComposites.bodyLarge       // 16px, Normal, leading 24px
typographyComposites.bodySmall       // 14px, Normal, leading 20px

// Labels
typographyComposites.labelLarge      // 14px, Medium, letter-spacing 0.1px
typographyComposites.labelSmall      // 12px, Medium, letter-spacing 0.5px
```

**Uso en estilos:**
```css
.heading-1 {
  font-family: var(--font-family-sans);
  font-size: var(--font-size-3xl);
  font-weight: var(--font-weight-bold);
  line-height: 1.2;
}
```

## 📏 Espaciado

Sistema de espaciado de **4px base** (múltiplos de 4):

| Token | Valor | Casos de uso |
|-------|-------|-------------|
| **0** | 0px | Sin espaciado |
| **xs** | 4px | Microgap entre elementos |
| **sm** | 8px | Espaciado mínimo dentro de elementos |
| **md** | 16px | Espaciado estándar (DEFAULT) |
| **lg** | 24px | Espaciado entre secciones |
| **xl** | 32px | Espaciado grande |
| **2xl** | 48px | Secciones mayores |
| **3xl** | 64px | Bloques principales |
| **4xl** | 96px | Espaciado muy amplio |

**Regla de oro:** Usa `md` como valor por defecto, sube/baja según necesidad.

```typescript
// React example
import { spacing } from '@/tokens';

<Box p={spacing.md} gap={spacing.sm} />
```

```css
/* CSS example */
.card {
  padding: var(--spacing-lg);
  margin-bottom: var(--spacing-xl);
  gap: var(--spacing-md);
}
```

## ⚡ Transiciones

Tiempos de animación para movimiento fluido:

| Token | Duración | Función | Uso |
|-------|----------|---------|-----|
| **fast** | 150ms | ease-in-out | Micro-interacciones |
| **base** | 200ms | ease-in-out | Interacciones estándar |
| **slow** | 300ms | ease-in-out | Transiciones notables |
| **slowest** | 500ms | ease-in-out | Animaciones complejas |

**Recomendaciones:**
- Hover effects: `fast`
- Page transitions: `base`
- Modal animations: `slow`
- Complejas: `slowest`

```typescript
import { transitions, getTransition } from '@/tokens';

// Acceso directo
transition: transitions.base;  // '200ms cubic-bezier(...)'

// Helper function
transition: getTransition('opacity', 'fast');
```

## 🌗 Dark Mode

Todos los tokens soportan dark mode automáticamente a través de:

```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: var(--color-neutral-900);
    --color-text-primary: var(--color-neutral-50);
    /* ... */
  }
}
```

**En React:**
```typescript
const isDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
const bgColor = isDark ? colors.neutral[900] : colors.neutral[0];
```

## 🚀 Cómo Usar

### En CSS

```css
/* Importar variables */
@import '@/styles/design-tokens.css';

.button {
  padding: var(--spacing-md) var(--spacing-lg);
  font-size: var(--font-size-base);
  background-color: var(--color-primary-500);
  color: var(--color-primary-900);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-md);
  transition: all var(--transition-base);
}

.button:hover {
  background-color: var(--color-primary-600);
  box-shadow: var(--shadow-lg);
}
```

### En React/TypeScript

```typescript
import { 
  colors, 
  spacing, 
  typography, 
  getCSSVariable,
  typographyComposites 
} from '@/tokens';

// Uso directo
const buttonStyles = {
  padding: `${spacing.md} ${spacing.lg}`,
  fontSize: typography.fontSize.base,
  backgroundColor: colors.primary[500],
  color: colors.primary[900],
  borderRadius: '8px',
  transition: `all 200ms ease-in-out`,
};

// Composiciones
const headingStyles = {
  ...typographyComposites.headlineLarge,
  color: colors.neutral[900],
  marginBottom: spacing.lg,
};

// Variables CSS
const dynamicBg = getCSSVariable('color-primary-500');
```

### En Styled Components / Emotion

```typescript
import styled from 'styled-components';
import { colors, spacing, transitions } from '@/tokens';

const Button = styled.button`
  padding: ${spacing.md} ${spacing.lg};
  background-color: ${colors.primary[500]};
  border-radius: 8px;
  transition: all ${transitions.base};

  &:hover {
    background-color: ${colors.primary[600]};
  }
`;
```

### En Tailwind CSS

Configura tokens como extensiones:

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: tokens.colors.primary,
        secondary: tokens.colors.secondary,
      },
      spacing: tokens.spacing,
      borderRadius: tokens.radius,
      boxShadow: tokens.shadows,
    },
  },
};
```
